General Streaming Support – What to Do Before You Call

This page outlines the basic technical checks you should complete before reaching out for live support about streaming issues.

For detailed Peacock TV installation and troubleshooting, see: Peacock TV Installation & Troubleshooting – Independent Help

1. Quick checks before contacting support

  1. Confirm other websites or apps work on the same device.
  2. Restart the streaming app and restart the device.
  3. Test another streaming service to see if the issue is app-specific.
  4. Run a quick speed test and note the download speed and ping.
  5. Check if the problem occurs on multiple devices on the same network.

2. Information to have ready when you talk to someone

  • Type of device (Smart TV, Firestick, Roku, phone, tablet, etc.).
  • Internet connection type (Wi-Fi / Ethernet / mobile hotspot).
  • Rough internet speed (from a recent speed test).
  • Whether the problem is buffering, not loading, sign-in failure, or error messages.
  • Any error codes you see on the screen.

When does it make sense to call for help?

If you’ve tried the basic steps above, confirmed your internet is working, and a streaming app still refuses to work correctly, it can be faster to talk to someone who can walk through the situation with you.
